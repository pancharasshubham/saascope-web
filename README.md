# SaaScope Web

SaaScope Web is the frontend application for the SaaScope API. It helps businesses manage their SaaS subscriptions, identify duplicate services, and find places where they are losing money on subscriptions they may not be using.

## What it does

- Accepts a CSV file containing SaaS subscription data.
- Sends the data to the SaaScope API for processing.
- Displays reports and insights about subscriptions.
- Shows duplicate SaaS services used by the business.
- Highlights subscriptions that may be wasting money or are no longer needed.

## Purpose

This frontend is designed to make it easy for users to upload subscription data and review a clear, visual report on their SaaS spending. It provides a user-friendly view of all subscription details so businesses can make better decisions.

## Key features

- CSV import for subscription data
- Report generation using SaaScope API
- Duplicate SaaS detection
- Cost and usage insights
- Detailed subscription overview
